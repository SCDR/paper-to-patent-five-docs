# 总复核

## 目的
在导出前完成最终一致性检查。

## 输入
- 五个已通过门禁的 Markdown 草稿
- 最终术语表、图号表、模块号表和公式清单
- Word 导出需求

## 输出
- 五份最终 Markdown 文稿
- 五份对应 Word 文档
- 总复核报告
- 未解决问题清单（若仍有）

## 文件输出
- `md/01_权利要求书.md`
- `md/02_说明书.md`
- `md/03_说明书附图.md`
- `md/04_说明书摘要.md`
- `md/05_摘要附图.md`
- 对应导出至 `docx/` 目录。

## 执行交叉检查
- 权利要求与说明书术语是否完全一致。
- 附图编号与说明书正文引用是否吻合。
- 摘要内容是否与说明书和实施例口径相符。
- 图号引用路径是否有效、资源是否可加载。
- 所有权利要求是否都能在说明书或实施例中找到支撑段落。

## 检查项
- 权利要求是否得到说明书支撑
- 摘要是否与附图一致
- 图号和模块标记是否一致
- 五个文件中的术语是否稳定统一
- 背景文献是否符合专利现有技术写法
- 最终 Word 文档是否保留专利排版和章节顺序
- 公式在 Word 中是否仍保持 LaTeX 对应关系且没有零宽字符

## 最终质量门禁
- 格式完整：五份文稿结构符合专利标准，无明显排版错误。
- 语义清晰：专业人士无需反复阅读即可理解技术方案。
- 可直接进入人工法务润色和提交准备，无需大幅重写。
- Word 中公式正确显示、无零宽字符，且与 LaTeX 形式一致。
