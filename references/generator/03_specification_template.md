# 说明书模板

目标文件：`md/02_说明书.md`

```
说明书
{{INVENTION_NAME}}

技术领域
{{TECH_FIELD_TEXT}}

背景技术
{{BACKGROUND_TEXT}}
例如文献一中，{{PRIOR_ART_1}}。
例如文献二中，{{PRIOR_ART_2}}。
现有技术问题包括：
1) {{PAIN_POINT_1}}；
2) {{PAIN_POINT_2}}；
3) {{PAIN_POINT_3}}。

发明内容
本发明的目的在于：{{CORE_PROBLEM}}。
为实现上述目的，采用如下技术方案：{{SOLUTION_OVERVIEW_MATCHING_CLAIM_1}}。
本发明有益效果为：{{BENEFITS_WITH_RESULTS}}。

附图说明
图1为{{FIG1_TITLE}}；
图2为{{FIG2_TITLE}}；
...

具体实施方式
实施例1：{{EXAMPLE_1}}
实施例2：{{EXAMPLE_2}}
实施例3：{{EXAMPLE_3}}
...
```

约束：
- 标题顺序固定：技术领域、背景技术、发明内容、附图说明、具体实施方式。
- 说明书必须逐条支撑权利要求。
- 文字部分可有公式/表格，不插图。
