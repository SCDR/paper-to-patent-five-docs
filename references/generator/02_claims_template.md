# 权利要求书模板

目标文件：`md/01_权利要求书.md`

```
{{INVENTION_NAME}}

1. 一种{{SUBJECT}}，其特征在于，包括：
   {{CLAIM_1_STEP_1}}；
   {{CLAIM_1_STEP_2}}；
   ...；
   {{CLAIM_1_STEP_N}}。

2. 根据权利要求1所述的{{SUBJECT}}，其特征在于，{{CLAIM_2_DETAIL}}。
3. 根据权利要求{{REF_ID}}所述的{{SUBJECT}}，其特征在于，{{CLAIM_3_DETAIL}}。
...
N. 根据权利要求{{REF_ID}}所述的{{SUBJECT}}，其特征在于，{{CLAIM_N_DETAIL}}。

式（1）：{{FORMULA_1_LATEX}}
式（1）中，{{FORMULA_1_SYMBOL_DESC}}
```

约束：
- 仅使用阿拉伯数字顺序编号。
- 每一项权利要求仅句末一个句号。
- 禁止“如图所示/如说明书所述”等引用。
- 可出现公式或表格，但不插图。
