# 全局参数模板

在生成任何文档前先填充以下参数，后续各模板复用：

- 发明名称：`{{INVENTION_NAME}}`
- 申请类型：`{{PATENT_TYPE}}`（发明/实用新型）
- 技术领域：`{{TECH_DOMAIN}}`
- 应用场景：`{{SCENARIO}}`
- 核心技术问题：`{{CORE_PROBLEM}}`
- 核心技术方案要点：`{{CORE_SOLUTION_POINTS}}`
- 关键效果数据：`{{KEY_RESULTS}}`
- 主要用途：`{{MAIN_USAGE}}`
- 术语表：`{{TERMS_DICT}}`
- 图号映射：`{{FIGURE_MAP}}`
- 模块编号映射：`{{MODULE_MAP}}`
- 公式清单（LaTeX）：`{{FORMULA_LIST}}`
- 参考文献映射：`{{PRIOR_ART_MAP}}`
