# 说明书摘要模板

目标文件：`md/04_说明书摘要.md`

```
说明书摘要
{{INVENTION_NAME}}
涉及{{TECH_DOMAIN}}，用于{{SCENARIO}}。
本发明针对{{CORE_PROBLEM}}，提出{{CORE_SOLUTION_POINTS}}。
通过{{KEY_METHOD_CHAIN}}，实现{{KEY_EFFECT}}。
实验或实施结果表明，{{KEY_RESULTS}}。
本发明主要用于{{MAIN_USAGE}}。
摘要附图指定为图{{ABSTRACT_FIG_NO}}。
```

约束：
- 摘要正文不设置分节小标题。
- 默认控制在 300 字以内；国际进入国家阶段场景可放宽。
- 必须覆盖：技术领域、技术问题、技术方案要点、主要用途。
