# 说明书摘要

## 目的
将说明书压缩为专利风格摘要。

## 输入
- 已通过门禁的说明书草案
- 已冻结的图号说明
- 需要保留的关键效果数据

## 输出
- 400~500 字摘要草案
- 代表附图引用
- 摘要与说明书一致性检查结果

## 编写规则

### 摘要长度与内容
- 控制在约 400~500 字。
- 保留四个要素：技术问题、核心方案（通常 3 个关键点）、关键效果数据、代表附图引用。
- 去除冗余步骤展开和无关细节。
- 突出与现有技术差异和创新性。

### 风格要求
- 中文主写法，核心术语首次出现时可括注英文。
- 避免营销化夸大词汇（如"革命性"、"突破性"）。
- 避免过度学术化表述（如"本文详细阐述"）。
- 采用直接陈述的专利风格。

## 门禁
- 摘要必须与说明书和附图口径一致。
- 摘要必须包含技术问题、核心方案、效果和代表附图引用。
