# 摘要附图 格式标准

一、申请发明专利或者实用新型专利应当提交摘要附图，一式一份。

二、申请人应当在说明书附图中指定一幅最能说明该发明或者实用新型技术特征的附图作为摘要附图。摘要附图应当是说明书附图中的一幅。

三、摘要附图首页用此页，续页可使用同样大小和质量相当的白纸。纸张只限使用正面，四周应当留有页边距：左侧和顶部各25毫米，右侧和底部各15毫米。

四、摘要附图的绘制要求与说明书附图相同：

1. 应当使用包括计算机在内的制图工具绘制，线条应当均匀清晰、足够深，不得涂改；
2. 大小及清晰度应当保证在该图缩小到三分之二时仍能清晰分辨出图中的各个细节；
3. 图中除必要的词语外不得有其他注释；
4. 附图标记应当使用阿拉伯数字编号，与说明书附图中的标记一致。

五、摘要附图应当在每页下框线居中位置顺序编写页码。

**说明**：摘要附图仅有一幅，是从说明书附图中精选出来最能代表发明核心技术方案的一幅图，通常选择系统整体架构图或核心流程图。
